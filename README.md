# Segment Anything Model (SAM) Image Segmentation

This repository contains code for **image segmentation using Meta's Segment Anything Model (SAM)**.  
Built as part of the **Adobe Mock PS**.

## How to Use
1. Open this script in Google Colab.
2. Upload:
   - The SAM model checkpoint (`sam_vit_h.pth`)
   - Any image you want to segment.
3. Run all cells to generate and visualize segmentation masks.

## Model
Download the official checkpoint from:
[https://dl.fbaipublicfiles.com/segment_anything/sam_vit_h_4b8939.pth](https://dl.fbaipublicfiles.com/segment_anything/sam_vit_h_4b8939.pth)
